# پروژه پیام‌رسان بلاخره (Balakhare)

بلاخره یک پلتفرم پیام‌رسان جامع، آنی و پیشرفته است که با الهام از تلگرام و بله، با استفاده از تکنولوژی‌های مدرن مایکروسافت طراحی و پیاده‌سازی شده است. این پروژه نمایشی از توانمندی در پیاده‌سازی سیستم‌های توزیع شده و آنی (Real-time) با استفاده از SignalR و معماری تمیز (Clean Architecture) است.

## قابلیت‌های کلیدی و پیشرفته

### پیام‌رسانی و تعامل
- **پیام‌رسانی آنی**: تبادل پیام‌ها در کسری از ثانیه با استفاده از SignalR.
- **انواع گفتگو**: پشتیبانی کامل از گفتگوی شخصی (PV)، گروه‌ها، کانال‌های اطلاع‌رسانی و اتاق گفتگو عمومی.
- **ابزارهای تعاملی**: قابلیت پاسخ (Reply)، فوروارد هوشمند، و حذف پیام (Unsend).
- **مدیریت محتوا**: قابلیت سنجاق کردن (Pin) پیام‌های مهم در هر گفتگو.
- **واکنش‌ها (Reactions)**: امکان ثبت لایک و واکنش‌های مختلف روی هر پیام.
- **نشانگر تایپ**: نمایش وضعیت در حال نوشتن (Typing Indicator) برای کاربران.
- **وضعیت آنلاین**: نمایش وضعیت برخط بودن کاربران به صورت زنده.
- **شمارنده پیام‌های خوانده نشده**: نمایش تعداد پیام‌های جدید در لیست گفتگوها.

### محتوا و رسانه
- **ارسال فایل**: پشتیبانی از ارسال تصاویر و فایل‌های مختلف با محدودیت حجم ۱۰ مگابایت.
- **پیش‌نمایش هوشمند لینک**: استخراج خودکار متادیتا (عنوان، توضیحات، تصویر) و نمایش کارت پیش‌نمایش برای لینک‌های وب.
- **جستجوی سراسری**: قابلیت جستجو در لیست گفتگوها و کاربران در سایدبار.
- **جستجوی داخلی**: امکان جستجوی متنی در تاریخچه هر گفتگو.

### مدیریت و تنظیمات
- **پنل مدیریت پیشرفته**: داشبورد مدرن برای نظارت بر آمار سیستم (تعداد کاربران، پیام‌ها، تعاملات) و مدیریت کاربران (مسدودسازی).
- **مدیریت پروفایل**: ویرایش نام، بیوگرافی و تصویر پروفایل توسط کاربر.
- **تم‌های مختلف**: پشتیبانی از حالت تاریک (Dark Mode) و روشن با ذخیره‌سازی وضعیت در مرورگر.
- **فونت فارسی**: استفاده از فونت استاندارد Vazirmatn برای بهبود تجربه کاربری.

## ساختار پروژه (Architecture)

این پروژه بر اساس اصول **Clean Architecture** در سه لایه اصلی پیاده‌سازی شده است:

### ۱. Balakhare.Core
لایه مرکزی شامل موجودیت‌های دامین (Entities)، انوم‌ها (Enums) و منطق پایه کسب‌وکار.
- `Entities/`: شامل User، Chat، ChatMessage، MessageReaction و ...
- `Enums/`: انواع چت (گروه، کانال، خصوصی).

### ۲. Balakhare.Infrastructure
لایه زیرساخت که مسئول تعامل با دنیای بیرون و مدیریت داده‌هاست.
- `Data/`: شامل `AppDbContext` (EF Core) و `DataSeeder` برای داده‌های اولیه.
- `Services/`: پیاده‌سازی سرویس‌های فایل و استخراج متادیتای لینک.
- **پشتیبانی از دیتابیس‌های مختلف**: SQLite (پیش‌فرض)، SQL Server، PostgreSQL و MySQL.

### ۳. Balakhare.Web
لایه نمایش و رابط کاربری (API و SignalR).
- `Hubs/`: مدیریت ارتباطات آنی با SignalR.
- `Controllers/`: نقاط اتصال API برای مدیریت حساب، چت و پنل مدیریت.
- `wwwroot/`: رابط کاربری ساخته شده با HTML، CSS (Tailwind) و JavaScript خام.

## راهنمای نصب و اجرا

### روش اول: اجرای سریع (Setup Script)
۱. ابتدا مخزن را کلون کنید.
۲. دستور زیر را برای آماده‌سازی خودکار اجرا کنید (کپی تنظیمات و نصب وابستگی‌ها):
```bash
chmod +x setup.sh
./setup.sh
```
۳. برنامه را اجرا کنید:
```bash
dotnet run --project src/Balakhare.Web
```

### روش دوم: استفاده از Docker
برای اجرای کل پلتفرم به همراه دیتابیس PostgreSQL در کانتینر:
```bash
docker-compose up --build
```

### روش سوم: دستی
۱. وارد پوشه `src` شوید.
۲. فایل `.env` را بر اساس `.env.example` بسازید و نوع دیتابیس را تعیین کنید.
۳. دستورات زیر را اجرا کنید:
```bash
dotnet build
dotnet run --project Balakhare.Web
```
برنامه در آدرس http://localhost:5000 (یا پورتی که در خروجی کنسول نمایش داده می‌شود) در دسترس خواهد بود.
