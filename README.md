# پروژه پیام‌رسان بلاخره

بلاخره یک پلتفرم پیام‌رسان جامع، آنی و پیشرفته است که با هدف نمایش تسلط بر ابزارهای اینترپرایز دات‌نت طراحی شده است.

## قابلیت‌های کلیدی

- پیام‌رسانی آنی: با استفاده از SignalR برای ارسال پیام، تایپ و وضعیت آنلاین.
- انواع گفتگو: پشتیبانی از PV، گروه‌های تیمی، کانال‌های خبری و چت‌روم عمومی.
- فیچرهای پیشرفته: سنجاق کردن پیام، پاسخ (Reply)، فوروارد، لایک و واکنش.
- جستجوی هوشمند: جستجو در محتوای پیام‌های هر گفتگو.
- پیش‌نمایش لینک: نمایش خودکار اطلاعات سایت‌های لینک شده.
- پنل مدیریت: داشبورد اختصاصی برای مدیریت کاربران و مشاهده آمار سیستم.
- دیتاسیدر (Data Seeding): پر کردن خودکار دیتابیس با داده‌های فرضی در اولین اجرا.

## ساختار پروژه (Project Structure)

پروژه از سه لایه اصلی تشکیل شده است:

- Balakhare.Core: شامل موجودیت‌ها (Entities) و انواع داده (Enums).
- Balakhare.Infrastructure: شامل لایه داده (DbContext)، دیتاسیدر و سرویس‌های پیش‌نمایش لینک و فایل.
- Balakhare.Web: شامل هاب‌های SignalR، کنترلرهای API، پنل مدیریت و رابط کاربری فرانت‌آند.

## راهنمای نصب و راه‌اندازی (Setup Guide)

۱. اطمینان از نصب .NET 8 SDK.
۲. کپی کردن پروژه و رفتن به پوشه src.
۳. تنظیم نوع دیتابیس در فایل appsettings.json. به صورت پیش‌فرض از SQLite استفاده می‌شود.
۴. اجرای دستورات زیر:
```bash
dotnet build
dotnet run --project Balakhare.Web
```
۵. پس از اجرا، دیتابیس به صورت خودکار ساخته شده و با داده‌های فرضی (کاربران admin، reza، sara) پر می‌شود.
