# پروژه پیام‌رسان بلاخره

بلاخره یک پلتفرم پیام‌رسان پیشرفته و آنی است که با استفاده از تکنولوژی‌های مدرن مایکروسافت (ASP.NET Core 8 و SignalR) طراحی شده است.

## قابلیت‌های کلیدی و پیشرفته

- گفتگوهای پیشرفته: پیام‌رسانی دو نفره، گروه‌ها، کانال‌ها و اتاق گفتگو عمومی.
- ابزارهای تعاملی: قابلیت پاسخ (Reply) به پیام‌ها و فوروارد کردن پیام بین گفتگوها.
- سنجاق کردن پیام (Pin): امکان سنجاق کردن پیام‌های مهم در بالای محیط گفتگو.
- واکنش به پیام‌ها: ثبت لایک و واکنش‌های مختلف به صورت آنی.
- جستجوی داخلی: قابلیت جستجوی متنی سریع در تاریخچه هر گفتگو.
- پیش‌نمایش هوشمند لینک: استخراج خودکار متادیتا و نمایش کارت پیش‌نمایش برای لینک‌های وب.
- مدیریت رسانه: ارسال تصویر و فایل تا سقف ۱۰ مگابایت.
- تم تاریک و روشن: طراحی مدرن و واکنش‌گرا با پشتیبانی از حالت شب.

## معماری و تکنولوژی

- Backend: ASP.NET Core 8, SignalR, Entity Framework Core.
- Database: پشتیبانی از SQLite, SQL Server و PostgreSQL.
- Frontend: HTML5, CSS3, Tailwind CSS, JavaScript.
- Security: بررسی سطح دسترسی و عضویت در هاب سیگنال‌آر.

## راهنمای راه‌اندازی

۱. تنظیم دیتابیس در appsettings.json (تعیین DatabaseProvider و ConnectionString).
۲. اجرای دستورات زیر در مسیر پوشه src:
```bash
dotnet build
dotnet run --project Balakhare.Web
```
