# مستندات فنی پلتفرم بلاخره

این سند شامل جزئیات فنی مربوط به ساختار پایگاه داده، متدهای هاب و پنل مدیریت سیستم بلاخره است.

## ساختار پایگاه داده (Database Schema)

### ۱. جدول کاربران (Users)
شامل اطلاعات پایه، وضعیت آنلاین بودن و فیلدهای مدیریتی (IsAdmin, IsBlocked).

### ۲. جدول گفتگوها (Chats)
شامل انواع گفتگو: شخصی، گروهی، کانالی و عمومی.

### ۳. جداول پیام‌ها، واکنش‌ها و اعضا
مدیریت محتوا، تعاملات و روابط بین کاربران و گفتگوها.

---

## پنل مدیریت (Admin Panel)

سیستم دارای یک پنل مدیریت اختصاصی است که از آدرس `/admin.html` در دسترس است. قابلیت‌های این پنل عبارتند از:
- مشاهده آمار کلی سیستم (تعداد کاربر، پیام، چت).
- لیست کامل کاربران ثبت‌نام شده.
- قابلیت مسدود کردن یا رفع مسدودیت کاربران.
- مدیریت گفتگوهای ایجاد شده.

---

## متدهای هاب و API

### ادمین API
- GET /api/admin/stats: دریافت آمار سیستم.
- GET /api/admin/users: لیست کاربران.
- POST /api/admin/users/{id}/block: تغییر وضعیت مسدودیت کاربر.
